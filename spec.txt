1. Executive Overview

The DEEPLY Hospitality Executive Dashboard is a fully integrated Power BI intelligence layer for the hospitality
sector. It unifies data from every operational system into a single management cockpit that connects commercial
performance, profitability, operations, people, and sustainability into one coherent view.

1.1 The Guiding Principle

  Data  Insight  Financial Impact  Recommended Action

  The dashboard is not another BI report. It is a management control layer that enables fast,
  data-driven, cross-departmental decision-making -- available in Hebrew and English.

1.2 Vision Statement

"Build for DEEPLY an advanced hospitality dashboard product at Enterprise level that looks and functions as a
strategic management system -- not a collection of reports." Every screen answers one management question.
Every KPI explains the why. Every anomaly leads to an action.

6. Data Model -- Full Schema

The model follows a Star Schema pattern. All Fact tables connect exclusively to Dimension tables. No direct
Fact-to-Fact relationships. Role-playing dimensions are implemented via inactive relationships activated with
USERELATIONSHIP() in DAX.

  Key Conventions:
   PK = Primary Key (surrogate integer, never exposed to end users)
   FK = Foreign Key referencing a Dimension table
   NK = Natural / Business Key (from source system)
   All date columns store as DATE or DATETIME, never VARCHAR
   All monetary columns: DECIMAL(18,4) in base currency
   NULL = allowed only where value genuinely may not exist
